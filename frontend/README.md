# Frontend

As instrucoes completas para rodar o projeto estao no README da raiz:

- [README.md](/home/af/dev/LAB03/README.md)

Resumo rapido:

```bash
cd frontend
npm ci
npm run dev
```

Se estiver rodando fora do Docker, use `frontend/.env.local` com:

```env
VITE_API_URL=http://localhost:3000
```
